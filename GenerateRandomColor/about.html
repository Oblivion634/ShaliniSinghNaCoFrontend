About the Website

The Generate Random Color project is an interactive web tool designed to explore and experiment with colors dynamically. It allows users to:

  • ✅ Generate random colors and display them in a table.
  • ✅ Experiment with RGB and CMY color models.
  • ✅ View hexadecimal and float-point representations of colors.
  • ✅ Explore web-safe colors with ease.

Built with HTML, CSS (Bootstrap), and JavaScript, this project offers a responsive and engaging experience for designers, developers, and color enthusiasts.
